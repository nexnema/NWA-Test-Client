Dark and Light mode

Normal Mode: Get theme mode from browser, Change system theme and see effect.
Light Mode: Force light mode by add nwaThemeLight to html tag
Dark Mode: Force dark mode by add nwaThemeDark to html tag

Color Themes

Normal Mode: Standard web colors
See BlackGold color by add nwaColorBlackGold to html tag
See PurpleGreen color by add nwaColorPurpleGreen to html tag
